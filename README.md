# Mendeleev

A minimalistic,Focused high-performance desktop notes app for Windows and (MacOs to be updated).
* Create, edit, and delete notes
* Sorting notes based on ( Date Created, Last modified, Title Name)
* Full Serach Across all notes
* Rich text Formatting
* Zen mode ( Fully focused Distraction free Note writing mode )
* Export indivijual notes as .txt
* Export/Import Full App Backup
* Light and Dark Theme

## Tech Stack
------
Built with Electron + React + Vite + Tailwind CSS. sql.js for local Storage. No cloud, no auth.

## Setup 
------
1. Install dependencies:
   npm install

2. Check in development mode:
   npm run dev

3. Build for Production:
   npm run build

4. Package into distributable app:
  npm run package:win #for windows - The file will be stored in releases directory
